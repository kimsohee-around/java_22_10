전역속성

전역속성은 모든 태그 요소가 공통적으로 갖는 속성들입니다.
참고 : MDN WEB DOCS

class 와 id

css 와 자바스크립트에서 사용하게 될 주요 전역속성입니다.

속성 설명 비교 기호
class class 전역 특성은 공백으로 구분한 요소 클래스의 목록으로, 대소문자를 구분하지 않습니다. 클래스는 CSS나 JavaScript에서 클래스 선택자나 DOM 메서드의 document.getElementsByClassName() (en-US)과 같은 메서드를 통해 요소에 접근할 수 있는 방법입니다. 여러 요소에 공통적으로 같은 값을 사용할 수 있고, 클래스를 여러개 적용 할수 있습니다. 클래스 이름을 지정할 때는 . 기호와 사용
id id 전역 특성은 문서 전체에서 유일한 고유식별자(ID)를 정의합니다. 고유식별자의 목적은 프래그먼트 식별자(4fragmentEx.html)를 사용해 요소를 가리킬 때와 스크립트 및 스타일 적용 시 특정 요소를 식별하기 위함입니다. DOM 메서드의 document.getElementsById() 로 요소에 접근. 특정 요소를 식별하기 위한것으로 일반적으로 중복된 값은 사용하지 않습니다. 그러나 사용해도 오류발생 하지 않습니다. id 이름을 지정할 때는 # 기호와 사용
